## Naming Convention

Documents follow this naming pattern:

```
YYYY-MM-DD-<tool>-<topic>-<type>.md
```

Where:
- **YYYY-MM-DD**: Creation date
- **tool**: `vauxhall` (Bigend), `praude` (Gurgeh), `tandemonium` (Coldwine), `intermute`, or general
- **topic**: Brief topic description with hyphens
- **type**: `design` (architecture/rationale) or `plan` (implementation steps)
